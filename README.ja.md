# Simple Easy Social Login

Simple Easy Social Login は、WordPress サイトに高速かつ簡単なソーシャルログイン機能を追加できる、軽量でユーザーフレンドリーなプラグインです。

**Google、Facebook、LinkedIn（無料）** に加え、  
**Naver、Kakao、Line（プレミアム）** のログインをサポートしており、  
日本・韓国・中国などのアジア地域向けサイトはもちろん、ヨーロッパや南米のユーザーにも適した設計となっています。

本プラグインは、WordPress 標準のログイン／新規登録ページとスムーズに連携し、  
WooCommerce のログインおよび新規登録フォームにも対応しています。  
ソーシャルプロフィール画像は、WordPress ユーザープロフィールのアバターとして自動的に同期されます。

また、**拡張可能な Provider アーキテクチャ**を採用しており、  
必要に応じて新しい OAuth Provider を独立した Add-on プラグインとして追加することが可能です。

---

## ✨ 主な機能

- Google ログイン（無料）
- Facebook ログイン（無料）
- LinkedIn ログイン（無料）
- Naver ログイン（プレミアム）
- Kakao ログイン（プレミアム）
- Line ログイン（プレミアム）
- ユーザーアバターの自動同期
- メールアドレスによる既存 WordPress ユーザーの自動連携
- ログイン／ログアウト／新規登録後のリダイレクト URL 設定
- Provider 設定用のシンプルで分かりやすい管理画面 UI
- ショートコード対応： [se_social_login]
- WordPress のログイン／新規登録フォームへの自動表示
- WooCommerce ログイン／新規登録フォーム対応（オプション）
- WordPress コーディング標準に準拠した軽量構造
- 不要なデータベーステーブルを作成しません
- 新しい OAuth Provider を Add-on プラグインとして拡張可能な Provider システム

## 🐞 デバッグログ

SESLP には、OAuth およびソーシャルログインの問題を診断するための  
組み込みデバッグログ機能が含まれています。

詳細なログの説明は、WordPress 管理画面から確認できます：
**SESLP → Guides → Debug Log & Troubleshooting**

ログファイルの保存先：

- `/wp-content/SESLP-debug.log`
- `/wp-content/debug.log`（`WP_DEBUG_LOG` 有効時）

## 🚀 インストール方法

1. プラグインを `/wp-content/plugins/simple-easy-social-login/` ディレクトリにアップロードします。
2. WordPress 管理画面の **プラグイン → インストール済みプラグイン** から有効化します。
3. **設定 → Simple Easy Social Login** に移動します。
4. 各ソーシャルログイン Provider の Client ID と Client Secret を入力します。
5. 設定を保存します。
6. フロントエンドでソーシャルログインボタンが正しく表示されることを確認します。

## ❓ よくある質問

### WooCommerce と一緒に使用できますか？

はい。WooCommerce のログインおよび新規登録フォームと連携します。

### WooCommerce のログインは動作しますが、リダイレクトの挙動が異なります。これは正常ですか？

はい。WooCommerce が有効な場合、ログイン後は通常 **マイアカウント** ページへリダイレクトされます。  
リダイレクト先の URL は、プラグイン設定または利用可能なフィルターでカスタマイズできます。

### WooCommerce サイトでソーシャルログインが動作しない場合、何を確認すればよいですか？

以下の点をご確認ください：

- WooCommerce が最新の安定版に更新されているか
- ソーシャルログインのプロバイダーがプラグイン設定で有効になっているか
- Client ID と Client Secret が正しく入力されているか
- リダイレクト / コールバック URL が各プロバイダーの開発者コンソールに正しく登録されているか
- カスタムログインやチェックアウトテンプレートが WooCommerce のデフォルトフックを削除していないか
- デバッグログが有効化され、`/wp-content/SESLP-debug.log` を確認しているか

### Google ログインのみ使用できますか？

はい。各 Provider は個別に有効化／無効化できます。

### プレミアムライセンスはいつ必要ですか？

**Naver、Kakao、Line** のログインを使用する場合は、プレミアムライセンスが必要です。  
Google、Facebook、LinkedIn は無料で利用できます。

### ショートコードはありますか？

はい。以下のショートコードを使用して、任意の場所にソーシャルログインボタンを表示できます。
[se_social_login]

### ユーザーアバターは自動で取得されますか？

はい。Google や Facebook など一部の Provider では、プロフィール画像を自動的に取得し、WordPress のアバターとして同期できます。
